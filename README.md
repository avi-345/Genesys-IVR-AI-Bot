# This is Basic chat BOT which gives an idea on how we can connect to any model using API's..

## Genesys IVR AI Bot
An AI-powered IVR routing system that uses LLM-based intent detection to route customers to the right department — built by someone who has spent 3+ years building these systems in Genesys professionally.

## What it does
Detects customer intent from natural language
Routes to correct department automatically
Maintains full conversation memory
Handles escalation to human agent
Generates AI session summary on exit
Handles no-input scenarios gracefully

## Departments supported
Balance & Payments
Tax Forms (W2, 1099)
Account Management  
Human Escalation
General Support

## Tech Stack
Python
Groq API (Llama 3.3 70b)
python-dotenv

I was using Groq API for this as it was free and give us an idea on how we can interact with models.
