# This is Basic chat BOT which gives an idea on how we can connect to any model using API's..

## Genesys IVR AI Bot
AI-powered IVR routing system using LLM intent detection.
Based on detected intent within the same Agent, will get routed to particular department.
Built with Python + Groq + Llama 3.3

## Features
- Intent-based routing
- Multi-department support  
- Conversation memory
- Escalation handling
- Session summary
